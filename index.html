Portfolio

Here are some notable projects from work and school I was involved in. Click on the icons to view the Github repo if the project has one!

Engineering Design Robotics Competition

After designing the physical and electrical pieces of our robot, the microcontroller was coded in Java to execute various movement and measurement tasks to save salamanders.

Surf Helmet

When a professor asks the research team to make a helmet for surfers to wear to analyze water conditions at various global locations, we built the helmet, as well as a notebook to visualize the data.

Text Mining in Python

Exploring the essentials of Python and data visualization using this text-mining tutorial notebook running on Jupyter Notebook.

Frame Diffference Camera Synthesizer

In a creative computing course, this project was designed to take a frame difference of a webcam, which could be used to manipulate a chorus-modulated saw-wave synthesizer.

Clowder Extractor Testing

SMU explored the Clowder framework for automating data extraction in a future system which would replace the current HPC Cluster ManeFrame 2.

Used Console Listing Aggregator (in progress)

In my software engineering principles course, my group has designed a webapp which would aggregate online listings for used gaming consoles from various sites. We are currently in development of this project to include listings from Ebay, Amazon, and Newegg using their APIs.
